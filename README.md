📌 Project Overview
The Enterprise Credit Card Approval System is a machine learning-based application that predicts whether a customer's credit card application should be approved or rejected. The system analyzes customer details such as income, employment status, credit history, and other financial factors to make an intelligent decision.
This project helps banks and financial institutions automate the credit approval process and reduce manual errors.

🎯 Objective
The main objective of this project is to:
Predict credit card approval using machine learning algorithms.
Improve decision-making in financial institutions.
Reduce risk by analyzing customer financial data.

⚙️ Technologies Used
Python
Streamlit (for web application interface)
Pandas & NumPy (for data processing)
Scikit-learn (for machine learning models)
Plotly (for data visualization)

🤖 Machine Learning Algorithms Used
The following algorithms are used to train and evaluate the model:
Logistic Regression
Random Forest Classifier
These algorithms help classify whether the credit card application will be Approved or Rejected.

🔄 Project Workflow
Data Collection
Data Preprocessing
Feature Engineering
Model Training
Model Evaluation
Prediction
Deployment using Streamlit

📥 Input Data
The system takes the following customer information as input:
Age
Income
Employment Status
Credit History
Number of Family Members
Housing Type
Other financial details

📤 Output
The system predicts:
Credit Card Approved ✅
Credit Card Rejected ❌
This helps banks quickly decide whether to approve a credit card application.

📊 Features of the Project
User-friendly web interface
Real-time prediction
Data visualization dashboard
Machine learning-based decision system

🚀 How to Run the Project
Clone the repository
Install required libraries
pip install -r requirements.txt
Run the Streamlit application
streamlit run app.py
Open the local browser to view the application.
